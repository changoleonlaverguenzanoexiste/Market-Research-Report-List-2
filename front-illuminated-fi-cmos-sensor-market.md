<p><strong>The Front illuminated FI CMOS Sensor Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Passive Pixel Sensor CMOS</li><li>Active Pixel Sensor CMOS</li></ul></p>
<p><p>Front illuminated FI CMOS Sensor market is divided into two types: Passive Pixel Sensor CMOS and Active Pixel Sensor CMOS. Passive Pixel Sensor CMOS technology uses a photodiode for each pixel, resulting in lower noise and higher sensitivity. On the other hand, Active Pixel Sensor CMOS technology integrates a photodiode, amplifier, and A/D converter in each pixel, allowing for faster readout and better noise performance. Both types offer unique benefits, catering to different requirements in the market.</p></p>
